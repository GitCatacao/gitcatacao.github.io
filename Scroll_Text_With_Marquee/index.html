How you can scroll text on the screen with Marquee HTML tag

It is a very simple method to make text scroll on the screen using only HTML without resorting to JavaScript or CSS animations.

Here some examples

marquee default: behavior=default_is_scroll loop=default_is_infinite (scrollamout_is_speed) marquee behavior="alternate" marquee behavior="slide" loop="0" (one shot, then stop) marquee behavior="scroll" scrollamount="50" loop="5" marquee behavior="scroll" direction="right" marquee behavior="alternate" direction="up" marquee behavior="scroll" scrollamount="20" direction="up" marquee behavior="scroll" direction="down" height="40px"
